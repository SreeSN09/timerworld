# ⏱ TimerWorld

A beautiful timer website built with Python (Flask) featuring a live local clock, stopwatch with lap tracking, and real-time world clocks for 12 cities across the globe.

---

## 🌍 Features

- **Local Clock** — Shows your current time, date, and timezone live
- **Stopwatch** — Start, Pause, Reset, and record Laps with a full lap history table
- **World Clocks** — 12 cities updating every second with a circular ring animation and day/night background tint
- **Beautiful UI** — Dark space theme with glowing clocks, animated background, and smooth hover effects
- **Fully Responsive** — Works on mobile and desktop
